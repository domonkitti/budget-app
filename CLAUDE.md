# budget-app

Next.js frontend for the budget app.

## Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Playwright (testing)

## Run locally
```bash
cp .env.local.example .env.local
npm run dev
```

## API
Backend at NEXT_PUBLIC_API_URL (default http://localhost:8080/api/v1)
See budget-app-api/CLAUDE.md for full API routes.
